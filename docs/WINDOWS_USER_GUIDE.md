# Windows 用户使用指南

## ⚠️ Windows 常见问题

### 问题1: 双击后无响应，窗口不显示

#### 可能原因
1. 杀毒软件拦截
2. 权限不足
3. 进程已在后台运行

#### ✅ 解决方法

**方法1: 以管理员身份运行（推荐）**
1. 右键点击应用图标
2. 选择"以管理员身份运行"
3. 点击"是"确认 UAC 提示

**方法2: 检查任务管理器**
1. 按 `Ctrl+Shift+Esc` 打开任务管理器
2. 查找 "Cursor续杯工具" 或 "Cursor.exe"
3. 如果存在，结束任务
4. 重新运行程序

**方法3: 添加杀毒软件白名单**
1. 打开 Windows Defender（或其他杀毒软件）
2. 添加应用到白名单
3. 重新运行

---

### 问题2: 提示"进程崩溃：launch-failed"

#### 可能原因
- 缺少运行库
- 应用文件损坏
- 系统权限问题

#### ✅ 解决方法

**步骤1: 安装 VC++ 运行库**

下载并安装 Microsoft Visual C++ Redistributable:
- [VC++ 2015-2022 (x64)](https://aka.ms/vs/17/release/vc_redist.x64.exe)

**步骤2: 重新下载安装**
1. 完全卸载当前版本
2. 删除残留文件：
   - `C:\Users\你的用户名\AppData\Roaming\Cursor续杯工具`
   - `C:\Users\你的用户名\AppData\Local\cursor-renewal-tool`
3. 重新下载并安装

**步骤3: 关闭其他 Electron 应用**
- 关闭 VS Code
- 关闭其他基于 Electron 的应用
- 重新运行

---

### 问题3: 管理员权限问题

#### 为什么需要管理员权限？

Windows 版本需要管理员权限来：
1. 更新注册表中的机器码（`HKLM\SOFTWARE\...`）
2. 修改系统文件
3. 确保文件操作成功

#### 如何获取管理员权限？

**方法1: 右键以管理员身份运行**（每次）

**方法2: 设置始终以管理员身份运行**（永久）
1. 右键应用图标 → 属性
2. 切换到"兼容性"标签
3. 勾选"以管理员身份运行此程序"
4. 点击"确定"

## 🚀 正确的使用流程

### Windows 10/11 用户

1. **关闭 Cursor 应用**
2. **右键应用图标 → 以管理员身份运行**
3. **输入授权码并验证**
4. **点击一键续杯**
5. **等待3-5秒**（优化后很快！）
6. **完成！打开 Cursor**

## 💡 Pro 提示

### 加快启动速度
1. 关闭不必要的后台程序
2. 将应用添加到杀毒软件白名单
3. 使用 SSD 硬盘

### 避免错误
1. 始终以管理员身份运行
2. 换号前完全关闭 Cursor
3. 等待操作完成（不要中途关闭）
